# Problema de Negócio

O mercado de placas de vídeo (GPUs) apresenta constantes mudanças relacionadas a preços, evolução tecnológica e lançamento de novos modelos.

Este projeto tem como objetivo analisar dados de GPUs para compreender a variação dos preços, evolução dos produtos e comportamento dos fabricantes no mercado.

Através do dashboard desenvolvido no Power BI, é possível analisar:

- Evolução dos preços das GPUs;
- Participação dos fabricantes;
- Quantidade de modelos disponíveis;
- Características técnicas dos produtos;
- Tendências do mercado.

O objetivo é transformar dados históricos em informações estratégicas para auxiliar análises e tomadas de decisão.
